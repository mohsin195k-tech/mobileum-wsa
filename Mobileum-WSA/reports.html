Highlight Report

Highlight report provides data across all live roaming partners: Traffic Volumes, TAP Charges and Discounted Charges, Average TAP Prices and

Average Post-Discounted Price per Service Type. This is the most widely used report that all Commercial and Finance Manager customer will find valuable as it provides an overview on agreement performance and accrual.

Forecast Report

This report will generate Traffic Volumes and Discounting Charges forecasted for the End of an Agreement on a per Agreement basis,

as well as Discount Achieved: Forecasted Traffic Volume, Agreement, Forecasted Charges, Average IOT Charge

Agreement Performance Report Credit Note

This report shows Discount Achieved (in Home currency) for all active IOT Rebate Agreements. It gives Gross settlement value for Payable

and Receivable positions.. Reports can be generated cumulatively for all the services or per Service Type (Voice, SMS or Data (+ VoLTE).

Agreement Performance Report Credit Note Single Month

This report shows Discount Achieved (in Home currency) for all active IOT Rebate Agreements based on the selected period by the user.

It gives a monthly accrual status of the selected agreement.
